README Editor Tool

Keybinds are shown in the game manager for placing objects
Keybinds for the camera are found in the camera object

The edit_mode_ boolean toggles editing where you can change the map width and height and start placing tiles
The default save key is left ctrl
currently only 1 map can be saved at a time though there is a backup if the map needs replacing

README Game

player can move the camera around and rotate it
player can only place objects when a round is over
default round start key is space
